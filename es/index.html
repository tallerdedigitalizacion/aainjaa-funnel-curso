FAQ

Preguntas frecuentes

Respuesta corta, clara y lista para pulirse despues con informacion comercial definitiva.

Cuando recibo el acceso?

Tras validar el pago, el backend podra aprobar el acceso y enviar instrucciones o activar el area privada.

En que idioma esta el contenido?

La web funciona en espanol e ingles. El contenido del producto se puede indicar por pack o en la respuesta del backend.

Necesito experiencia previa?

No necesariamente, aunque el nivel recomendado actual es intermedio. Este copy es editable por idioma.

Puedo pagar con Stripe?

Si. El formulario esta preparado para redirigir a checkout o payment link cuando el backend o la configuracion lo indiquen.

Hay opcion de pago manual?

Si. La respuesta del backend puede devolver instrucciones y un codigo de referencia para seguimiento.

El producto incluye material fisico?

Solo si el pack configurado lo requiere. En ese caso se muestra direccion de envio y se informa de disponibilidad.

Cuanto tarda la entrega?

El acceso digital suele ser rapido tras validacion. Si hubiera material fisico, los tiempos son orientativos y pueden variar por logistica o aduanas.

Que es private access?

Es el placeholder de la futura entrada con contrasena al contenido o area exclusiva de alumnos.
